# 🎣 Phishing Awareness Analysis System

## 📌 Project Overview

Phishing is one of the most common cyberattacks used by attackers to steal sensitive information such as passwords, banking details, and personal data.

This project analyzes messages, emails, and URLs to identify phishing indicators and alert users about potential threats.

The system scans the provided text for suspicious keywords, malicious links, and common phishing patterns.

---

# 🎯 Objective

To develop a phishing detection system that:

- Detects suspicious keywords
- Identifies potentially harmful links
- Raises awareness about phishing attacks
- Improves cybersecurity understanding
